# Pycharm配置多运行脚本

在实际使用中，我们可能会需要以不同的参数运行同一份脚本文件，那么需要允许脚本并行运行。

1. 打开pycharm

2. 点击工具栏 `run->edit configurations`

   ![py_in_parallel](/py_in_parallel.png)

3. 例如我们需要对sagpool.py文件并行运行，则勾选allow running in parallel

   ![sagpool_in_parallel](/sagpool_in_parallel.png)
